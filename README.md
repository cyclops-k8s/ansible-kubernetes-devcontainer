# Intro

These are the source files for the [ansible-kubernetes](https://github.com/cyclops-k8s/ansible-kubernetes) Visual Studio Code [devcontainer](https://code.visualstudio.com/docs/devcontainers/containers).

The container is hosted on [Quay](https://quay.io/repository/cyclops-k8s/ansible-kubernetes-devcontainer).

The container is intended to be used by the Visual Studio Code [Dev Container extension](https://marketplace.visualstudio.com/items?itemName=ms-vscode-remote.remote-containers).

# How To Use

Refer to the code example in the [.devcontainer directory in the ansible-kubernetes git repo](https://github.com/cyclops-k8s/ansible-kubernetes/tree/main/.devcontainer) for a full example.
